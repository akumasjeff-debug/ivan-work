# PTT 留言彈幕 — 專案總覽

把 PTT 文章的推文變成「彈幕」的桌面小工具。有兩種形式：

## 1. 桌面覆蓋層版（主要版本）— `overlay.pyw`

- 彈幕直接飄在**桌面最上層**：背景全透明只看得到字、永遠置頂、滑鼠預設**穿透**（不擋任何操作）。
- 一個小的**控制視窗**選看板/文章、調參數；控制視窗縮到最小，彈幕照飛。
- **調整位置/大小**：按鈕進入調整模式 → 彈幕區浮出半透明框，拖曳移動、拉右下角改大小，位置大小會記住（存 `overlay-settings.json`）。
- 純 Python 內建 tkinter，**不需安裝任何套件**；由 `start.bat` 用 pythonw 啟動（無黑窗）。

## 2. 網頁版 — `danmaku.html` + `server.py`

- 瀏覽器 App 模式視窗（全螢幕舞台），由 `start-web.bat` 啟動（port 8003）。

## 共通功能

- **熱門看板列表**（含人氣）＋手動輸入任意看板名。
- **文章列表**：推文數/標題/作者、翻頁；刪除文與置底文自動略過。
- **從頭重播**：現有推文全部依序發射。
- **直播模式**：跳過舊推文只看新的（末 5 則暖場）。
- **常駐即時同步**：不管哪種模式，播放中都走 PTT 官方 longpoll 機制，新推文約 1 秒內自動接著播（不需再按任何鍵；機制細節見 HANDOVER）。
- 推＝綠、噓＝紅、→＝白；可調速度／字級／密度／不透明度、顯示/隱藏推文者 ID、暫停、清空。

## 檔案

| 檔案 | 用途 |
|------|------|
| `overlay.pyw` | 桌面覆蓋層版主程式（tkinter，含抓取/解析/彈幕引擎/控制視窗） |
| `overlay-settings.json` | 覆蓋層位置大小與偏好（自動產生） |
| `danmaku.html` | 網頁版前端（單一 HTML） |
| `server.py` | 網頁版本機伺服器：靜態檔 + `/ptt/*` 代理到 ptt.cc（繞 CORS） |
| `start.bat` | 啟動桌面覆蓋層版（無黑窗） |
| `start-web.bat` | 啟動網頁版（server + App 模式視窗） |
| `make-shortcut.bat` | 在桌面建「PTT Danmaku」捷徑指向 start.bat（跑一次即可） |
| `HANDOVER.md` | 交接細節 |
