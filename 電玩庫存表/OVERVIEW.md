# 電玩庫存表 — 專案總覽

PChome 電玩商品的庫存＋週銷管理，本體是一份 **Google 試算表**，
本機腳本一鍵把每週兩個下載檔轉進 13 個分頁。

## 試算表

- 網址：https://docs.google.com/spreadsheets/d/1KTfQSanQiciSfIM5QF2UHKXbBtHa9fGS1DfK_Quevl4/edit

| 分頁 | 內容 |
|------|------|
| 庫存表 | pchome採購（7777）快照，stock 版面（匯出檔原順序） |
| 傑仕登、宏碁遊戲 | 含庫存廠商頁：該廠商全部商品（含經 pchome採購歸戶的），stock 版面，週銷大→小 |
| 壹世代、瑋琳、誠碁、銀科互通、玥勝、智冠、寶可夢台灣、英益誠＋艾格瑞 | 純銷量廠商頁：品項＋週銷＋業績＋毛利（無庫存欄），全品項含週銷 0，週銷大→小；本週有賣但不在匯出檔的商品列在後面註記 |
| 可賣量歷史 | 一列一商品、一欄一匯入日期，看庫存變化 |
| 週銷歷史 | 一列一商品、一欄一週（07/13-07/19 格式），看銷量趨勢 |

**stock 版面 12 欄**：首欄（廠商名稱/館名稱）｜商品編號｜商品名稱｜可賣量｜成本｜售價｜來源廠商｜週銷（含日期區間）｜可售週數｜業績金額｜毛利額｜毛利率
警示色：**深紅**＝缺貨但本週有銷量（最優先補）；淺紅＝缺貨；橘＝庫存撐不到一週（可售週數<1）。

## 每週更新流程

1. PChome 後台下載兩個檔到 Downloads：
   - 全站廠商商品匯出 CSV（`20########_XXXX.csv`）——已含 pchome採購，舊的採購專用 .xls 不用再下載
   - 即時業績（`MMDD-MMDD即時業績.xlsx`）
2. `py 電玩庫存表\匯入轉換.py` → 全部分頁自動更新，完成

## 廠商歸戶規則

「來源廠商」有值用它（採購品的真實供應商），否則用「廠商編號」（寄倉品）。
兩邊檔案同一套規則，所以傑仕登頁會同時包含直屬寄倉與經 pchome採購進貨的商品。

## 檔案

| 檔案 | 用途 |
|------|------|
| `匯入轉換.py` | 兩個下載檔 → 組 13 分頁 payload → POST GAS Web App 寫入；payload 備份存 `reports/` |

| `config.local.json` | Web App 網址＋token（gitignore 僅本機） |
